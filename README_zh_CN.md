**python-lekin**是一个APS智能排产调度工具。考虑实际约束的前提下，实现动态调整计划排程，高效响应客户订单承诺。

- 支持工艺路线约束
- 支持产能约束
- 支持物料齐套约束
- 支持顺排、倒排等排产方法
- 支持遗传算法排产
- 支持强化学习排产

``` shell

```

### 使用

``` python
from lekin import Heuristics, Genetics
from lekin import Scheduler

solver = Heuristics()
scheduler = Scheduler(solver)
scheduler.solve(jobs, machines)

scheduler.draw()
```
